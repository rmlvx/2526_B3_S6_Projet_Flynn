# Projet_Flynn

**Nom de code**: FLYNN pour Follow Lines

**Objectif** : Concevoir est de mettre en place un robot capable de suivre des lignes noires ou blanches.

## Composants principaux

|Composants|Nombre|Fonctionnalités|Documentation|
|----------|------|---------------|-------------|
|Raspberry PI Zero 2W|1|Alimente et contrôle les composants|[RaspberryPi_Doc](https://www.raspberrypi.com/documentation/computers/raspberry-pi.html#raspberry-pi-zero-2-w)|
|LSM6DSOX|1|Mesure les coordonnées X,Y,Z dans l'espace et mesure la pesanteur|[LSM6DSOX_Doc](https://www.st.com/resource/en/datasheet/lsm6dsox.pdf)|
|Capteur TMC2225|2|Permet de diriger la direction des roues du robot avec un calcul|[TMC_Doc](https://www.analog.com/media/en/technical-documentation/data-sheets/TMC2225_datasheet_rev1.14.pdf)|
|Capteur MCP3208|1|Permet au robot de détecter les couleurs|[MCP3208_Doc](https://ww1.microchip.com/downloads/en/DeviceDoc/21298E.pdf)|

## Stack Technique

Conception du schéma et du PCB : ``Kicad 9.0``
