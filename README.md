# Projet_Flynn

Nom de code : FLYNN pour Follow Lines

Objectif : Concevoir est de mettre en place un robot capable de suivre des lignes noires ou blanches.

## Composants principaux

|Composants                 |Nombre|Fonctionnalités                                                               |
|:-------------------------:|:----:|:----------------------------------------------------------------------------:|
|Raspberry PI Zero 2W       |1     |Alimente et contrôle les composants                                           |
|LSM6DSOX                   |1     |Mesure les coordonnées X,Y,Z dans l'espace et mesure la pesanteur             |
|Capteur TMC2225            |2     |Permet de diriger la direction des roues du robot avec un calcul              |
|Capteur Infrarouge         |1     |Permet au robot de détecter les couleurs afin de savoir il peut avancer ou non|

- Rasperry PI ZERO 2W
- Accéléromètre LSM6SDSOX de STM32 Electronics
- 2 Capteurs TMC2225
- Capteur Infrarouge
